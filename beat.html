🎮 플레이 방법

화면 위에서 아래로 내려오는 노트를 판정선에 맞춰 정확한 타이밍에 처리하는 리듬 게임입니다.

기본 조작 각 레인에 대응하는 키를 눌러 노트를 처리합니다
기본 키 (4레인) DF / JK
기본 키 (5레인) DF / Space / JK
기본 키 (8레인) ASDF / JKL;
일시정지 ESC 또는 화면 우상단 버튼

💡 조작 키는 환경설정 → 조작 탭에서 자유롭게 변경할 수 있습니다. 레인 수에 따라 활성화되는 키가 달라집니다.

🎵 노트의 종류
일반 노트
판정선에 맞춰 한 번 누르면 처리됩니다. 가장 기본적인 노트입니다.
롱 노트
시작 지점에서 누른 채로 유지하다가 끝 지점에서 정확하게 떼야 합니다. 도중에 놓으면 MISS 처리됩니다.
가짜 노트 함정
누르면 MISS! 키를 누르지 않고 통과시켜야 PERFECT를 받습니다. '어려움' 난이도에서 등장합니다.
동시타
같은 타이밍에 여러 레인에서 동시에 등장합니다. 대응하는 키를 동시에 눌러야 합니다.

콤보 초기화
콤보 GOOD 이상 연속 시 쌓이며, 높을수록 추가 점수가 올라갑니다. BAD·MISS는 콤보를 초기화합니다.
랭크 결과 화면에서 최종 점수에 따라 S / A / B / C 랭크가 부여됩니다.
온라인 점수 로그인 상태에서 온라인 차트를 클리어하면 점수가 리더보드에 자동 기록됩니다.
🕹️ 게임 모드
🎲
랜덤 모드
노트가 자동 생성됩니다. 난이도와 레인 수, 세부 파라미터를 원하는 대로 설정하세요.
🎼
뮤직 모드
직접 만든 차트(JSON)와 음악 파일을 불러와 정확한 박자에 맞춰 플레이합니다.
🌐
온라인 라이브러리
다른 사람이 업로드한 차트를 검색하고 바로 플레이하거나 리더보드를 확인합니다.
레인 수 4 ~ 8레인 선택 가능. 레인이 많을수록 더 넓은 손가락 운용이 필요합니다.
난이도 (랜덤) 쉬움 / 보통 / 어려움 선택 후 세부 설정으로 속도·확률을 직접 미세 조정할 수 있습니다.
⚙️ 난이도 세부 설정

랜덤 모드에서 세부 설정 패널을 열어 각 파라미터를 개별 조절할 수 있습니다.

낙하 속도 노트가 판정선까지 내려오는 속도입니다. 값이 높을수록 빠릅니다. (쉬움 4 / 보통 7 / 어려움 10)
생성 속도 노트가 얼마나 빠르게 연속으로 등장하는지를 결정합니다. (쉬움 1.0x / 보통 1.5x / 어려움 2.0x)
동시타 확률 여러 레인 노트가 동시에 등장할 확률입니다. (쉬움 10% / 보통 25% / 어려움 40%)
최대 동시타 수 한 번에 등장할 수 있는 동시타 최대 개수. 레인 수를 초과할 수 없습니다.
롱노트 확률 전체 노트 중 롱 노트가 등장하는 비율입니다. (쉬움 10% / 보통 15% / 어려움 20%)
가짜 노트 토글로 활성화하면 가짜 노트가 등장합니다. 기본적으로 어려움 난이도에서만 활성화됩니다.

💡 프리셋 버튼(쉬움/보통/어려움)을 누른 뒤 세부 값을 조금씩 조절하면 나만의 커스텀 난이도를 만들 수 있습니다.

✏️ 차트 에디터

자신만의 차트를 직접 제작하는 공간입니다. 음악 파일을 먼저 불러온 뒤 타임라인에 노트를 배치하세요.

기본 조작
노트 타입 선택 1일반 2롱노트 3가짜
레인 입력 키 QWERTYUIO
실행 취소 Ctrl+Z 최대 50단계
재생 / 정지 Space 또는 재생 버튼
에디터 기능
스냅 노트 배치 격자의 세밀도를 조절합니다. 4분음표 / 8분음표 / 16분음표 등을 선택하세요.
트리거 차트 중간에 BPM 또는 노트 낙하 속도를 변경하는 지점입니다. 타임라인에 노란 선으로 표시됩니다.
미리보기 재생 시 우측 게임 화면에서 노트 낙하를 실시간으로 확인할 수 있습니다.
저장 / 열기 차트를 .json 파일로 내보내거나 기존 파일을 불러옵니다.
클라우드 업로드 로그인 후 완성된 차트를 온라인 라이브러리에 공유할 수 있습니다.

💡 롱노트는 타임라인에서 시작 셀을 클릭하고 드래그하여 길이를 지정합니다. 노트 클릭으로 삭제할 수 있습니다.

🌐 온라인 기능
회원가입 / 로그인 우측 상단 계정 아이콘을 눌러 이메일로 가입하거나 로그인합니다.
라이브러리 탐색 온라인 버튼에서 다른 유저가 공유한 차트를 검색하고 바로 플레이할 수 있습니다.
리더보드 온라인 차트 클리어 시 점수가 자동으로 기록되며, 차트별 리더보드에서 순위를 확인할 수 있습니다.
차트 업로드 에디터에서 제작한 차트를 업로드해 다른 유저와 공유할 수 있습니다. 업로드 후 수정도 가능합니다.
닉네임 설정 계정 아이콘 → 프로필에서 리더보드에 표시될 닉네임을 설정합니다.
